NAME

Noembed::ImageSource - a base class for image sources

DESCRIPTION

This is a subclass of Noembed::Source meant for images. User's of this base class should not define a serialize method, but instead define an image_data method.

This class will automatically check for maxheight and maxwidth parameters in the request. If found, the image will be scaled down to fit the requested dimensions.

METHODS

image_data ($body)

Receives the downloaded content and must return a hashref containing src and title keys. These will be used to build the final HTML for the response.
