NAME

Noembed::Util - useful functions for Noembed

DESCRIPTION

This package includes a number of functions that are used throughout Noembed. Many of these are asynchronous and accept a callback as the last argument.

FUNCTIONS

http_get ($url, %options, $callback)

Download a URL and call the callback when it is completed. See AnyEvent::HTTP for a list of options.

  Noembed::Util::http_get $url, sub {
    my ($body, $headers) = @_;
    if ($headers->{Status} == 200) {
      ... do some work.
    }
  };
http_resolve ($url, $callback)

Determine what location, if any, a URL redirects to.

  Noembed::Util::http_resolve "http://bit.ly/abcd", sub {
    my $resolved = shift;
    ... do some work.
  };
colorize ($text, %options, $callback)

Syntax highlight a block of text. Valid options include: language, filename. See Noembed::Pygmentize for more options.

dimensions ($image_url, [$request,] $callback)

Download an image url and determine the height and width. If a Noembed::Request object is included this will check for the maxwidth and maxheight parameters and scale down the dimensions based on these limits.

html ($text)

Returns a version of $text that will not be automatically escaped when used inside a template.
