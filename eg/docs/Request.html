NAME

Noembed::Request - a client embed request

DESCRIPTION

This class represents a client embed request. See Plack::Request for details on most methods.

ADDITIONAL METHODS

hash

A SHA1 hash representing the requested embed.

url

The URL that the client requested for embedding. This should not change throughout the response.

content_url ($url)

The URL that Noembed will download. Typically, you will not need to change this. It is populated by a source's build_url method. However, it can be used in a pre_download hook to modify the URL will be downloaded.

pattern

If the requested URL matched any of the loaded Sources this will return the compiled regular expression that it matched.

captures

Returns an array reference with any captures from the matched pattern.
